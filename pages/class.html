== Class ==

It's a full featured module that aims to port the class logic of other OOP 
languages to javascript.



Current supported features are:


    
    
  • Abstract class: basic support to avoid instantiation
    • 
    
  • Final class: you can't extend a final class or add members dynamically
    • 
    
  • Base: it's the same of "extend" keyword of other langs
        your object inherits all methods of extended class and they shares
        prototype that allow you to cast the instanceof succesfully
    
    • 
    
  • Use: it's the mechanism of traits used also by other languages to  
        horizontally extends other class. in few words: 
        you can use methods from other objects. 
        It's not fully implemented yet, if you have some issues please
        report 
        
    
    • 
    
  • Private members
    • 
    
  • Public members
    • 
    
  • Static members
    • 
    
  • Final members
    • 



Work in progress features:


    
    
  • Protected methods: will be available soon[tm]
