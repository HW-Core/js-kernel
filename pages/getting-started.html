Getting started ( Sample Code ) 


After the installation , you can use following bootstrap code:

* if you want run it on client, it's the bootstrap for browser (index.html)


<html>
    <head>
        <title>My Index Page</title>
        <script type="text/javascript">
            function myFunction() {
                // you can write your code here using hw2core features
            }
        </script>
        <!-- data-after-boot is an attribute to define a javascript file or a function to run after kernel has been loaded -->
        <script data-after-boot="myFunction" src="upt_modules/hw2/modules/js/src/kernel/index.js"></script>
    </head>
    <body>

    </body>
</html>


* if you want run it on server, it's the bootstrap for nodejs (index.js): 


var Hw2Core = require("upt_modules/hw2/modules/js/src/kernel/index.js");

Hw2Core(function() {
    // you can write your code here using hw2core features
});
