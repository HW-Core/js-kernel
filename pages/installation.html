To install 


* requirements for installation ( not for library itself ):
nodejs 0.10.+ ( older not tested )
upt 1.4.16+

if you don't have upt, please install it using npm : 
npm install -g upt ( this command on linux requires to run with root permission )

execute this command in a terminal ( nodejs cmd on windows ):

upt install hw2-core/js-kernel


If you want just test it without install

use bootstrap code below changing:

HW2PATH_ROOT="./upt_modules/" -> HW2_PATHROOT="http://hw2-core.github.io/"

and

"upt_modules/hw2/modules/js/src/kernel/index.js" -> "http://hw2-core.github.io/hw2/modules/js/src/kernel/index.js" 

please don't use this url for production since it is not stable 
and github is not a CDN


SAMPLE CODE:


* if you want run it on client, it's the bootstrap for browser (index.html)


<html>
    <head>
        <title>My Index Page</title>
        <script type="text/javascript">
            HW2PATH_ROOT="./upt_modules/"; // url path where "hw2" folder is stored
            
            function myFunction() {
                var Hw2Core=this; // scope of this function
                // you can write your code here
            }
        </script>
        <!-- data-after-boot is an attribute to define a javascript file or a function to run after kernel has been loaded -->
        <script data-after-boot="myFunction" src="upt_modules/hw2/modules/js/src/kernel/index.js"></script>
    </head>
    <body>

    </body>
</html>


* if you want run it on server, it's the bootstrap for nodejs (index.js): 


HW2PATH_ROOT = "./upt_modules/"; // relative/absolute path where hw2 folder is stored

var Hw2Core = require("upt_modules/hw2/modules/js/src/kernel/index.js");

Hw2Core(HW2PATH_ROOT, function() {
    // you can write your code here
});
