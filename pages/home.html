== Kernel ==

This is the javascript kernel module for hw2-core ( our extremely modular framework )
It is required by all hw2-core js libraries since implements
the bootstrap process and basic functionalities.

Hw2-Core JS is crossplatform and can run on browser and server ( nodejs tested )


to install: upt install hw2-core/js-kernel

bootstrap for browser (index.html): 


<html>
    <head>
        <title>My Index Page</title>
        <script type="text/javascript">
            HW2PATH_ROOT="./"; // url path where "hw2" folder is stored
        </script>
        <!-- data-after-boot is an attribute to define a javascript file to run after kernel has been loaded -->
        <script data-after-boot="index.html.js" src="hw2/modules/js/src/kernel/index.js"></script>
    </head>
    <body>

    </body>
</html>


bootstrap for nodejs (index.js): 


HW2PATH_ROOT = "./"; // relative/absolute path where hw2 folder is stored

var Hw2Core = require("hw2/modules/js/src/kernel/index.js");
